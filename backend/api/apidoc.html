This Documentation is valid for API commit at 2.12.2011

API:

http://bastler.bplaced.net/osm/api

List Pois

Nessesary Parameters

bbox=<x-min><y-min><x-max><y-max>
it's a filter for an area specified by the bounding box

Optional Parameters

filter='<key>=<value>,<key>=<value>,...'
filter=<key>,<key>,...
filter='<key>,<key>=<value>,...'
finds only entries where one of this key=value pairs matches

who=<email>
an email address that I know who is using this API (its possible that I inform you on API changes)

Get POI Details

Nessesary Parameters

id=<id>
the ID is the OSM object ID. Returns all Information of this Object stored in OSM (except history)
